# 🐍 贪吃蛇游戏 - 增强版

## 🎮 新增特色功能

### 🔊 音效支持
- 吃食物音效
- 游戏结束音效  
- 背景音乐（可选）
- 所有音效可单独开关

### 🎨 主题系统
- **经典主题**: 绿色蛇身 + 红色食物
- **霓虹主题**: 彩色渐变蛇身 + 发光食物
- **暗黑主题**: 深色背景 + 亮色元素
- **自定义主题**: 通过配置文件修改颜色

### ⚙️ 游戏设置菜单
- 音量调节（主音量、音效、音乐）
- 控制灵敏度调整
- 网格显示开关
- 自动保存设置

### 🎯 多种游戏模式
- **经典模式**: 标准贪吃蛇规则
- **无尽模式**: 速度不会增加，挑战极限长度
- **生存模式**: 随机障碍物，考验技巧
- **计时模式**: 在限定时间内获得最高分

### 📊 改进的统计系统
- 详细游戏统计（总游戏次数、平均分数等）
- 成就系统（首次游戏、高分突破等）
- 分数历史记录

```
├── README_IMPROVED.md        # 详细使用说明

```

## 🚀 使用方法

### 直接运行
```bash
python snake_game_improved.py
```

### 打包成可执行文件
```bash
python build_improved_game.py
```

### 自定义配置
编辑 `game_config.json` 文件：
```json
{
    "theme": "classic",
    "sound_enabled": true,
    "music_enabled": false,
    "volume": {
        "master": 1.0,
        "sfx": 1.0,
        "music": 0.7
    },
    "grid_visible": true,
    "initial_speed": 10,
    "speed_increase_interval": 3
}
```

## 🎮 完整操作指南

### 游戏中控制
- **方向键**: 控制蛇的移动方向
- **空格键**: 暂停/继续游戏
- **ESC**: 打开设置菜单
- **M**: 返回主菜单
- **R**: 重新开始当前游戏

### 主菜单选项
- **开始游戏**: 选择游戏模式
- **设置**: 调整音效、主题等
- **统计**: 查看游戏历史和成就
- **退出**: 关闭游戏

### 设置菜单
- **音效**: 开关音效和调节音量
- **主题**: 切换视觉主题
- **显示**: 网格开关、全屏选项
- **控制**: 灵敏度调整
- **保存**: 保存当前设置

## 📱 跨平台支持
- **Windows**: 生成 `.exe` 文件
- **macOS**: 生成 `.app` 包  
- **Linux**: 生成可执行文件
- **所有平台**: 支持直接运行 Python 脚本

## 💡 技术特性
- **模块化设计**: 易于扩展新功能
- **配置驱动**: 通过 JSON 配置自定义游戏
- **资源管理**: 自动处理缺失的音效文件
- **错误恢复**: 异常情况下自动保存进度
- **性能优化**: 流畅的游戏体验

## 🛠️ 开发者指南

### 添加新主题
在 `assets/themes/` 目录下创建新的主题文件，或在 `game_config.json` 中直接定义颜色。

### 添加新音效
将音效文件放入 `assets/sounds/` 目录，支持 `.wav` 和 `.ogg` 格式。

### 扩展游戏模式
继承 `GameMode` 基类，实现新的游戏逻辑。
