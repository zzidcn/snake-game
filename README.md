# 🐍 贪吃蛇游戏 - 峰哥专属版

## 🎮 游戏特色
- **精美界面**: 彩色蛇身、动态食物、网格背景
- **完整功能**: 最高分保存、动态难度、暂停/继续
- **流畅体验**: 优化的性能和控制响应
- **本地化**: 中文界面，专为峰哥定制

## 📁 文件结构
```
├── SnakeGame_峰哥版          # 可执行文件（Linux）
├── snake_game_enhanced.py    # 游戏源代码
├── build_snake_game.py       # 打包脚本
├── README.md                 # 使用说明
└── high_score.json           # 最高分记录（自动生成）
```

## 🚀 使用方法
1. **直接运行**: 双击 `SnakeGame_峰哥版` 或在终端运行:
   ```bash
   ./SnakeGame_峰哥版
   ```

2. **游戏操作**:
   - **方向键**: 控制蛇的移动方向
   - **空格键**: 暂停/继续游戏
   - **ENTER**: 开始新游戏
   - **R键**: 游戏结束后重新开始
   - **M键**: 返回主菜单
   - **ESC键**: 退出游戏

3. **游戏规则**:
   - 吃到食物增加分数和长度
   - 每吃3个食物速度增加一级
   - 撞到自己身体游戏结束
   - 最高分会自动保存

## 📱 跨平台支持
- **Windows**: 需要重新打包生成 `.exe` 文件
- **macOS**: 需要重新打包生成 `.app` 文件  
- **Linux**: 直接使用当前可执行文件
